Ruby versions changelog


Ruby 2.5.x

2.5.0 (Dec 2017) - 2.5.3 (Oct 2018). Status: Supported

Implementations: MRI 2.5.3 JRuby 9.2.0.0

  • Added yield_self - unlike #tap, it returns the result of the block
  • Added rescue in blocks
  • Added Hash#slice and Hash#transform_keys
  • Added ERB#result_with_hash
  • Changed New Array aliases - #prepend (#unshift) and #append (#push)
  • Changed Pattern argument on Enumerable methods
  • Changed Keyword arguments on Struct.new
  • Changed Removed Top-level constant lookup

Ruby 2.4.x

2.4.0 (Dec 2016) - 2.4.5 (Oct 2018). Status: Supported

  • Added Enumerable#sum
  • Added Integer#digits
  • Added Regexp#match?, Regexp#named_captures
  • Changed New constructor Logger.new(STDOUT, level:, progname:)
  • Changed Float: #round, #ceil, #floor, and #truncate now accept a precision
  • Removed Removed Fixnum and Bignum classes (unified to Integer)

Ruby 2.3.x

2.3.0 (Dec 2015) - 2.3.8 (Oct 2018). Status: Security maintenance planned until March 2019

Implementations: MRI 2.3.8 9.1.17.0

  • Added Safe navigation operator &.
  • Added Frozen string literals
  • Added Array#dig
  • Added Hash#fetch_values, Hash#to_proc, Hash#dig
  • Added ''Did you mean?
